# 🌟 FlappyBot 2D 🌟

**FlappyBot 2D** is a Unity-based game inspired by the popular *Flappy Bird* game. Help the bot avoid obstacles and soar through the air! The game features 2D controls, animated characters, and smooth gameplay mechanics.

🛠️ **Built with**:
- **Unity Engine** (2018.4.24f1)
- **C# Programming**
- **Visual Studio Community Edition 2019**

All assets used in this project are free and available online, and the project is open for educational purposes. Feel free to tweak and build upon it!

---

## 🚀 Features

This project shows you how to implement the following features:

- **C# Programming**: Implement core mechanics and behaviors of the game.
- **2D Player Movement**: Move the bot up and down by clicking the mouse.
- **Background Music**: Add and manage background music in Unity.
- **Prefabs**: Create reusable game objects.
- **Random Object Spawning**: Spawn obstacles at random intervals.
- **Animation**: Animate objects using the Animator component.
- **UI Buttons & Text**: Design a main menu with buttons for interaction.
- **Button Functions**: 
  - **Start Game**: Begin the game when the "Play" button is clicked.
  - **Credits**: View the credits when the "Credits" button is clicked.
  - **Quit**: Exit the game with the "Quit" button.

## ⚙️ Installation Guide

Follow these simple steps to get started with the project:

1. **Install Unity** – Download and install Unity Engine from [here](https://unity.com/).
2. **Set Up Visual Studio** – Install Visual Studio, the recommended editor for Unity.
3. **Clone the Repository** – Clone this project from GitHub to your local machine.
4. **Open the Project** – Launch Unity and open the cloned project.
5. **Open Main Menu Scene** – Load the **MainMenu** scene to start.
6. **Edit Scripts** – Use Visual Studio to open and edit the C# scripts.
